CS240 Project 1

Simple Text Analyzer

Due Date: Monday 02/01/2010 11:59PM

Overview:

In this project, we'd like you to implement a small application which can do simple analysis and operations over a given text.

Task:

After your program is executed, a menu should be prompted, which indicates the operations that can be done by the analyzer:

1. Input Text
2. Text Stats
3. Find Substring
4. Reverse Text
5. Quit

User can enter '1'-'5' followed by a new line character. Other input should be considered as invalid and your program should output "Invalid Input". (eg. 6, 10, a,...)

Input Text

User can input text no more than 1000 characters. After the new text is entered, it overwrites the old one. The default text should be an empty string. Notice that the text doesn't include the last new line character. If the user enters more than 1000 characters, print out the warning message and let the user enter again.

Text Stats

The program outputs the statistical information about the text entered.

The information includes:

Character Number: The length of the text.

Word Number: The number of words in the text. In this case, a word only consists of alphabet (a-z,A-Z). So the string "it's" has two words ("it" and "s"), "4.5" contains 0 word.

Sentence Number: The number of sentences in the text. The sentence is defined by '.' or '!' or '?' followed by ' ' or '\0' or ''' or '"'. So the string "hello world" contains 0 sentence while "hello. World!" contains 2 sentences.

Most Frequent Characters: Three most frequent characters in the text. The program should output the characters and their frequencies in descending order. However, sometimes the number of output characters will be less than 3 or more than 3. For the string "aaaabb", there're only 2 different characters. In this case, just output 'a':4  'b':2.  For the string "ffffddccabb", all 'b' 'c' 'd' have the same frequency of 2. In this case, output 'f':4  'b':2  'c':2  'd':2. If characters have the same frequency, output them in ascending order by ASCII value.

Find Substring

After user selects this option, he will input a substring no more than 1000 characters. Then program outputs the times the substring occurs in the original text. For example, if the text is "howdoyoudo?" and the substring is "do", there're 2 occurrences. Overlapping should be taken into account. For example, if the text is "aaaabbaa" and the substring is "aa", the occurrence time should be 4 instead of 3.

Reverse Text

Reverse the word in the text. In this case, a word is regarded as continuous characters without ' ' and '\t' in it. So "it's" is one word, "http:www.purdue.edu" is also one word. If the original text is "   i'm here, mom", the resulting text should be "mom here, i'm    ".

The Big Bang is the leading theory that almost all astrophysicists believe explains the origin of the universe. This is because all observations so far made support the Big Bang theory; there are four main lines of evidence that are most-often used. The first was discussed above: The expansion of the universe. The universe is expanding now, so in the past it must have been smaller. If it were smaller in the past, then there probably was a time when it was infinitesimally small. One could ask why don't we think that it might be expanding now but it could have been shrinking before and we just don't know about it. The answer is that there is simply no mechanism that we know about that could accomplish this transition on a universal scale. The second line of evidence is the Cosmic Microwave Background Radiation (CMB) that was discovered in 1965 by Arno Penzias and Robert Wilson from Bell Labs. They were working with a microwave receiver used to communicate with the Telstar satellite, but were getting noise from every direction they pointed the receiver. It was coming from all over the sky at what seemed to be exactly the same frequency. This was the first evidence for the CMB, and they later shared a Nobel Prize for this discovery. The CMB is an "echo" left over from when the universe was approximately 300,000 years old, as predicted by the Big Bang model. As something becomes compressed, as matter was when the universe was young, it becomes hot. The actual "heat" comes from particles' movements - the faster they move, the more energetic they are, and so the more heat we see. The universe was so hot before it was 300,000 years old that atoms could not form. Because of this, photons - particles of light - could not move around, for they kept reacting with electrons - the negatively charged parts of atoms. Therefore, during this period, the universe was effectively opaque. Once the universe had reached 300,000 years old, atoms could form, and electrons were now bound to a nucleus. Once this happened, photons could move about freely. This "first light" is the CMB, and its existence is a very strong indication that the Big Bang occurred. The third major pillar of the Big Bang theory lies in the abundance of the different elements of the universe. The theory predicts that certain amounts of hydrogen (~72%), helium (~28%), and other elements should be made. Observations have shown almost exactly the amounts that are predicted. The fourth piece is that the Big Bang theory is the only one that comprehensively lays down a framework for the eventual evolution of the universe as we observe it today.
